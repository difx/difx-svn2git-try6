.vex File Editor

The Vex File Editor allows you to locate, create, view and edit .vex files.  At least in theory - not much has been done yet.  Moreover, this documentation is crap.

Generating a New .vex File

There are a number of ways you can generate a .vex file, either as a final product or a template for hand-editing.  These include:

  1. Getting the file associated with the observations for your experiment, either directly or by using sked.  This is probably the most common way of doing this, as it is the easiest.
  2. Copying the file from another experiment.
  3. Grabbing one from a location on the DiFX Host.
  4. Filling in all of the information by hand using the editor.  This is probably the most error-prone, not to mention painful method.
