Pass Editor

The Pass Editor is used to view and change the settings associated with a single pass.  A pass is a collection of "jobs" which are generated with a single run of the DiFX program vex2difx.  Most of the settings available in the Pass Editor end up in a single DiFX .v2d file.  This file is used by vex2difx to make adjustments when converting a .vex file into DiFX .input files, from which jobs can be run.
